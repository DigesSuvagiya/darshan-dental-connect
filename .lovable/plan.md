
# Darshan Enterprise — Premium Dental Supply Landing Page

A fast, mobile-first single-page site optimized for QR-scan → instant contact.

## Visual Style
- Palette: white background, soft medical blue (#E8F1FB / #2563EB accents), subtle grey text
- Typography: Inter (one modern sans-serif), strong hierarchy
- Generous spacing, rounded corners, soft shadows — clean healthcare feel
- No heavy animations; subtle transitions only on desktop

## Sections (top → bottom)

**1. Hero (signature)**
- Centered Darshan Enterprise main logo
- Brand logos arranged around it:
  - Desktop: circular ring layout with subtle floating animation
  - Mobile: ring is replaced by a clean horizontal scroll strip (no animation)
- Business name "Darshan Enterprise" — bold, modern
- Tagline: *"Everything Dentists Need, In One Place"*
- Primary WhatsApp CTA below tagline (thumb-zone)

**2. About**
- 2–3 line trust statement about the business
- Three small trust chips (e.g. Authorized Distributor • Fast Delivery • Trusted by Clinics)

**3. Contact (main focus)**
- Three large, full-width, rounded buttons stacked on mobile, row on desktop:
  - WhatsApp (highlighted green, #25D366)
  - Call (solid blue)
  - Email (outline)
- Address line + working hours below

**4. Brands**
- Full grid of 15+ brand logos (2 cols mobile / 4–6 cols desktop)
- Soft grayscale logos, color on tap/hover
- Clean alignment, breathable spacing

**5. Footer**
- Logo, copyright, quick contact repeat

## Persistent Element
- **Floating WhatsApp button** bottom-right, always visible, minimal pulse-free design

## Mobile-First Rules Applied
- Vertical stacking, large 56px+ tap targets, no overlapping elements
- Horizontal scroll for hero brand ring on mobile (not floating animation)
- Body text ≥ 16px, comfortable line height

## Performance
- Pure HTML/CSS/Tailwind, no heavy libraries
- Logos as optimized PNG/WebP, lazy-loaded below the fold
- Single page, no routing overhead — sub-1s load target

## Content I need from you before building
- Main logo PNG + 15+ brand logos
- WhatsApp number, phone number, email, address, hours
